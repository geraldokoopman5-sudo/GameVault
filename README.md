A small React project built for upskilling and practicing modern React development concepts.

GameVault is a frontend-only web application that allows users to manage and organize their personal video game collection in one place.
Instead of opening multiple game launchers such as Steam, Epic Games, or other platforms, users can use GameVault as a personal library to view and keep track of their games.
The goal of this project is to practice React fundamentals while building a realistic application.

JavaScript
HTML5
CSS3
